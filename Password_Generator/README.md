# 🔐 Password Generator

A simple Python application that generates secure random passwords using letters, numbers, and special characters.

## Features

- Generate random passwords
- User can choose password length
- Includes uppercase and lowercase letters
- Includes numbers
- Includes special characters

## Technologies Used

- Python 3

## How to Run

```bash
python password_generator.py
```

##  Example

Input:
```
Enter password length: 8
```

Output:
```
Generated Password: A@7kP2#m
```
